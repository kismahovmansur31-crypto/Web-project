Описание проекта: Polaroid Album
Веб-приложение на Python для хранения персональных фотографий с эффектом полароида.
Функционал:
Регистрация и авторизация пользователей.
Ограничение объема: максимум 6 фотографий на один аккаунт.
Автоматическая обработка: наложение белой рамки полароида и текстовой подписи при загрузке.
Интерфейс: небесно-голубой фон с сеткой из 6 слотов. Пустые места обозначены белым пунктирным контуром.
Стек технологий:
Бэкенд: Python (Flask).
Обработка изображений: Pillow (PIL).
База данных: SQLite.
Фронтенд: HTML, CSS (Grid/Flexbox).
Логика работы:
Программа проверяет количество записей в базе данных для текущего пользователя. Если лимит (6 фото) не исчерпан, доступна форма загрузки. Загруженное изображение обрабатывается скриптом, преобразуется в формат полароида и отображается в соответствующей ячейке сетки на главной странице.
